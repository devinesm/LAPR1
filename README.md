# Volunteer Social Network

Academic project developed during the 1st year of the Computer Engineering degree at ISEP, created by a team of four students. The goal was to conceptualize and present a social network designed to connect volunteers with initiatives and organizations.

## Project Description

The project includes a mobile app interface designed in Figma and a simple website created to showcase the main idea. The focus was on design, user experience, and the basic structure of a digital product.

## Project Structure

- **website/**: Website developed using HTML, CSS, and Bootstrap  
- **app-interface/**: Mobile app prototype created in Figma
  
## Technologies Used

- Figma  
- HTML, CSS, Bootstrap  
- Git/GitHub for version control

## Team

Project developed by a team of four ISEP students as part of a 1st-year course.
